docplex.cp.config.py module¶

Configuration of the CPO Python API

This module is the top-level handler of the configuration parameters for the Python API to CPO. It contains the default values of the different configuration parameters.

It should NOT be changed directly. The preferable way is to add at least one of the following files that contain the changes to be performed:

  • cpo_config.py, a local set of changes on these parameters,
  • cpo_config_<hostname>.py, a hostname dependent set of changes.

Final set of parameters is obtained by loading first this module, then cpo_config.py and finally cpo_config_<hostname>.py. These modules should be visible from the PYTHONPATH and are loaded in this order to overwrite default values.

If called as main, this module prints the configuration on standard output.
