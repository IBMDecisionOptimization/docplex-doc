Known limitations¶

Limitations for DOcplex.MP¶

  • Some models cannot be expressed currently with DOcplex.MP. The following modeling constructs are not available yet:
    • piecewise linear functions,
    • logical connectors (and, or, not),
    • truth value of constraints, and
    • all callbacks other than progress callbacks.

Limitations for DOcplex.CP¶

  • Constraint programming models can only be solved with CP Optimizer on the cloud (DOcplexcloud).
