Mathematical Programming Modeling for Python (DOcplex.MP)¶

Welcome to the Mathematical Programming Modeling for Python feature of IBM® Decision Optimization CPLEX® Modeling for Python, (DOcplex.MP).

With this library, you can quickly and easily add the power of optimization to your application. You can model your problems by using the Python API and solve them on the cloud with the IBM® Decision Optimization on Cloud service or on your computer with IBM® ILOG® CPLEX Optimization Studio.

You can read more about IBM® Decision Optimization on Cloud here and its free trial version here.

You can read more about IBM® ILOG® CPLEX Optimization Studio here and its free Community Edition here.

This API is licensed under the Apache License, Version 2.0, and is numpy/pandas friendly.
