docplex.mp.environment module¶

class docplex.mp.environment.Environment(start_auto_configure=True)[source]¶

Bases: object

This class detects and holds all information regarding other modules of interest, such as whether CPLEX, numpy, and matplotlib are installed.

cplex_location¶
Returns:The system path where CPLEX is located, if present. Otherwise, returns None.
Return type:string
has_cplex¶
Returns:True if the CPLEX libraries are available.
Return type:Boolean
has_matplotlib¶
Returns:True if the matplotlib libraries are available.
Return type:Boolean
has_numpy¶

Return type:Boolean
has_pandas¶

Return type:Boolean
