docplex.mp.quad module¶

class docplex.mp.quad.QuadExpr[source]¶

Bases: docplex.mp.basic.Expr

This class models linear expressions. This class is not intended to be instantiated. Quadratic expressions are built either using operators or using Model.quad_expr().

clone()[source]¶

Makes a copy of the quadratic expression and returns it.

Returns:A quadratic expression.
is_separable()[source]¶

Returns True if all quadratic terms are separable.

iter_quad_triplets()[source]¶

Iterates over quadratic terms.

This iterator returns triplets of the form v1,v2,k where v1 and v2 are decision variables and k is a number.

number_of_quadratic_terms¶

Returns the number of quadratic terms.

Counts both the square and product terms.

Examples

q1 = x**2 q1.number_of-quadratic_terms >>> 1 q2 = (x+y+1)**2 q2.number_of_quadratic_terms >>> 3
