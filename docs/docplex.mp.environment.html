docplex.mp.environment module¶

class docplex.mp.environment.Environment(start_auto_configure=True)[source]¶

Bases: object

This class detects and holds all information regarding other modules of interest.

It detects whether CPLEX, numpy, and matplotlib are installed.

cplex_location¶

Returns the system path where CPLEX is located, if present.

has_cplex¶

Checks whether the CPLEX libraries are available.

has_matplotlib¶

Checks whether matplotlib libraries are available.

has_numpy¶

Checks whether numpy libraries are available.

has_pandas¶

Checks whether pandas libraries are available.

is_64bit()[source]¶

Returns True if running on a 64-bit platform.
